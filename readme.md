# Realtime Chat App

A full-stack realtime chat application built by:

- **Samroz Burhan F2022266116**
- **Anas Mukhtar F2022266140**

## Features
- Realtime messaging (Socket.io)
- Authentication
- Cloudinary Image Uploads
- Admin Control Panel
- MongoDB Integration

## Getting Started
1. Clone the repo
2. Run `npm install`
3. Configure `.env`
4. Start with `npm run dev`
